== Description ==

This helpful plugin will help you:

* Show bigger image when customer hover your product image on list
* Customer will happy because image so good, good feeling
== Installation ==

1. Upload `hover-popup.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Make sure woocommerce installed and actived
4. That this!. Browse your product, hover on product image and happy!
